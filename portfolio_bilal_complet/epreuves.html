Mes Épreuves et Projets - BTS SIO SISR

E4 Création d’un VPN entre deux sites

Description : Mise en place d’un VPN sécurisé entre Marrakech et Montpellier pour connecter les deux agences de l’entreprise Adertechnologies.

Compétences :

  • Installation et configuration d’un serveur OpenVPN
  • Création et gestion des certificats avec Easy-RSA
  • Routage et accès sécurisé entre les réseaux

Outils utilisés : Ubuntu Server, Easy-RSA, OpenVPN, pfSense

E5 Installation d’un serveur Active Directory

Description : Déploiement d’un serveur Windows avec gestion des utilisateurs, des groupes et des droits d’accès via Active Directory.

Compétences :

  • Installation de Windows Server
  • Création de l'annuaire Active Directory
  • Configuration des GPO (stratégies de groupe)
  • Attribution de droits aux utilisateurs

Outils utilisés : Windows Server 2016, Active Directory, GPO

E5 Maintenance du réseau local

Description : Réorganisation du câblage réseau, installation de switchs et configuration des adresses IP fixes pour les postes clients.

Compétences :

  • Pose de goulottes murales et câblage RJ45
  • Configuration réseau (IP statique, DNS, passerelle)
  • Tests de connectivité et dépannage

Outils utilisés : Matériel réseau, Windows 10/11, câbles RJ45, testeurs
